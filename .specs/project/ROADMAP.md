# Roadmap

**Current Milestone:** MVP Auth + Permissões
**Status:** In Progress

---

## Milestone 1: MVP Auth + RBAC

**Goal:** Sistema funcional de autenticação e gerenciamento de permissões
**Target:** Funcional localmente

### Features

**Autenticação** - IN PROGRESS
- Login com email e senha
- Cadastro com email e senha
- Logout
- Sessão persistente via cookie

**App Shell** - PLANNED
- Sidebar inset com navegação
- Header com user menu
- Layout responsivo

**Gerenciamento de Permissões** - PLANNED
- CRUD de roles
- Atribuição de roles a usuários
- Visualização de permissões por role
- Proteção de rotas por role
